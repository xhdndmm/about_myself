## 个人经历
到这里是时候说一下我的经历了，下面的内容可能有些长，如果你不愿意看，可以跳到下一个话题。  
> 本章节部分内容可能与实际不符，因为我记不清具体内容了，以后会根据照片、聊天记录等修改

我大约在2019-2020年初步接触到了“电脑”这个东西，因为当时我想去我老爹公司看看（？），然后正好老爹公司发的笔记本他用不着，放了很久，于是乎就给我了。当时的我对这些没太深入研究，期间正好赶上了新冠疫情，封城在家带着。后来上过编程班（图形化编程，连python这种都不是，好像在少年宫上的）。

2022年的时候认识了一位专业知识相对丰富的网友，在他那里了解到了很多，直到现在也是。后来又有一个想法，去互联网上发短视频。其实我最开始在微信的视频号上发视频，可能是是没什么人看吧，我就到抖音上发布视频。那段时期的记忆不是很深刻了，总之比较好笑。因为啥都不懂的原因，~~经常在互联网上瞎扯~~。

2023年之后，我拥有了一台服务器，虽然都是二手捡垃圾来的，但是我用它学到了许多专业知识，比如linux操作系统的相关使用、公网访问服务器、网站搭建等等吧。这段时间我还学会了3d建模（虽然现在有些忘了），当时用的是3dmax这个软件，我记得我花了1个月左右学会了常规操作，能做一些基础的模型。当时的视频现在还能在B站看到，也是那段时期为开始在B站发布视频，同时删掉了我多年的抖音账号（因为感觉那里的氛围不够好）。其实那段时间还留级过，当然不可能是学习问题。在上面提到过，我有成骨不全症，简单来说就是先天性的骨质疏松吧，虽然我不是很严重，但是在大概2023年～2024骨折了三次，给我留下了极为深刻的印象。因为这三次都是腿骨折，所以就只能在家修养了。正是那段时期我在家里待着，我学到了很多。

2024年的时候组建了我的网站，同时那段时期AI开始出现，我有点怀疑我这个兴趣爱好的前景，但同时又发现AI的能力真的很强，并开始利用它。那年我还步入了初中，虽然累但是成绩还算可以，大概中等生水平。

2025年的时候组建了自己的博客，并且优化了我的网站。同时我开始尝试使用ArchLinux作为我电脑的操作系统，因为Windows11的体验非常差，有各种奇怪BUG,并且系统占用还很高。到现在我感觉ArchLinux还是很不错的，~~[必须打个广告](https://archlinux.org/)~~。并且我在Github做了一系列项目，~~虽然不少是用AI辅助的~~，但是感觉还是有利用价值的。因为我换了邮箱，所以以前的贡献不显示qwq。

2026年初认识了一个朋友，还教他搭建了自己的博客。同时开始学习Go语言，因为Python并不怎么够用。
